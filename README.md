#### Команды

``` bash
# Запустить программу в режиме разработки
yarn run tauri dev

# Запустить программу в режиме разработки без tauri
yarn run dev

# Сбилдить программу
yarn run tauri build

# Сгенирировать статический html (nuxt)
yarn run generate

```
